# 🚀 Pinak Datta — Developer Portfolio


## 📌 Project Overview
A modern, fully responsive **personal developer portfolio** built with **React 18**, **Framer Motion** animations, **React Router v6**, and **CSS Modules**. This portfolio showcases my skills, projects, education, and contact information with a polished dark UI and smooth scroll-triggered animations.

The portfolio is designed to impress recruiters and demonstrate real frontend engineering skills — not just a template, but a hand-crafted, production-quality web application.
